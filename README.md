# WL-Sentinel 电气速查工具 V1.0

## 1. 项目概述
本工具定位为纯前端静态网页应用，采用移动端优先设计，旨在为电气工程师提供现场快速查询与计算工程数据的能力。项目完全由单文件 HTML 组成，无需安装与后端支持，发送至微信等平台即可点开使用，加载速度快，且具备离线使用能力。

## 2. 技术栈
*   **核心框架**：Vue 3 (基于 CDN 全局引入，`vue.global.prod.min.js`)
*   **UI 样式**：Tailwind CSS (基于 CDN 引入，`tailwind.min.js`)
*   **部署方式**：单文件 `WL-Sentinel.html` 纯静态原生运行。

## 3. 核心功能模块 (七大场景)
1.  **铜排查询模块 (`copper`)**
    *   提供 21 种常用矩形铜排规格的理论重量、敞露/封闭环境交流载流量查询。
    *   支持单片、2片叠加、3片叠加查询（V1.0 中多片叠加载流量预留为“待补充”）。
    *   内置封闭/敞露载流量效率比自动计算与颜色预警（绿色>85%，黄色75-85%，红色<75%）。
2.  **型材重量查询模块 (`profile`)**
    *   全覆盖扁钢（63种）、圆钢、方钢、等边角钢、槽钢的理论重量查询。
    *   支持快速模糊搜索过滤。
3.  **电缆载流量查询模块 (`cable`)**
    *   一键切换“三芯电缆”与“四芯电缆”，涵盖 1.5~300mm² 的常用规格。
    *   支持多种型号（YJV, VV, YJV22, BVV）过滤。
4.  **变压器电流速查模块 (`transformer`)**
    *   覆盖 50kVA~1600kVA 共 15 种主流容量。
    *   高压侧默认为 10kV，低压侧支持 400V / 700V 自由切换，实时出电流。
5.  **不锈钢箱体计算模块 (`box`)**
    *   拆分为**展开面积计算**与**底座面积计算**双功能，全表格化呈现。
    *   支持材质选择（201/203/301/304）、板厚以及宽深高/底座尺寸输入。
    *   自动计算面积与理论总重量，支持输入单价以快速预估材料费。
6.  **电气常用换算模块 (`calc`)**
    *   支持 380V 三相系统下，有功功率(kW)快速计算出视在功率(kVA)与额定电流(A)。
    *   功率因数与设备效率支持手动微调（默认 0.80 与 0.95）。
7.  **UPS 续航计算模块 (`ups`)**
    *   **正向计算**：根据 UPS 功率、负载率、电池容量与节数，推算理论续航时间（内置 0.85 损耗系数）。
    *   **反向计算**：根据目标续航时间与负载，推荐单组电池容量与串联节数。

## 4. 全局交互与特性
*   **一键复制**：所有模块计算或查询的结果均配备“一键复制”按钮，结果为预组装的格式化文本，方便现场直接粘贴到微信沟通或记录备忘录。
*   **路由支持**：使用 `window.location.hash` 记录当前选中的功能模块（如 `#copper`, `#cable`），支持复制链接分享直达特定功能。
*   **自适应与防误触**：禁用双击缩放和移动端高亮，采用大尺寸点击区域以适应工地现场的手机操作。

## 5. 待续与后续升级方向 (V2.0预留)
1.  **数据字典补充**：铜排模块中 2片叠加 和 3片叠加 的载流量目前作为占位（显示待补充），需在后续版本中更新补全。
2.  **缓存记忆**：可引入 `localStorage` 保存用户设置的单价偏好、功率因数等微调参数，避免重复输入。
3.  **扩展数据支持**：可增加更多绝缘材料性能、型材机械性能等表格参考资料。

## 6. 数据字典维护指南
项目所有数据均硬编码在 `WL-Sentinel.html` 底部的 `<script>` 标签内。如需进行后期的数据更新或修正，请直接在文件中搜索并修改以下常量数组：
*   `copperData`：铜排基础数据（包含规格 `spec`、重量 `weight`、敞露/封闭载流量）。
*   `flatSteelSpecs` 及相关生成函数：用于生成各类钢材规格和重量。
*   `cableData`：电缆线径与不同环境载流量字典。
*   `transformerCapacities`：变压器支持的容量列表字典。
