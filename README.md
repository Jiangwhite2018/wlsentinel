# WL-Sentinel 电气速查工具 V1.0

## 1. 核心定位与技术要求
*   **工具类型**：纯前端静态单文件 HTML 应用，无后端依赖、无安装要求。
*   **技术栈**：基于 Vue 3 (CDN) + Tailwind CSS (CDN) 开发。
*   **适配场景**：移动端优先，兼容微信内置浏览器及主流手机浏览器，支持首次加载后离线使用，首屏加载≤3秒。
*   **视觉风格**：工程蓝（#3B82F6）为主色调，绿/红/黄为辅助色。轻量化设计（无多余动画），所有表格统一采用全边框样式（外边框+内边框），线条颜色/粗细/间距全域一致。
*   **命名规范**：全场景统一使用“WL-Sentinel”作为工具名称，覆盖页面标题、导航栏、功能页眉等，无别称、后缀或前缀。

## 2. 核心功能模块详细说明 (九大场景)
1.  **铝排查询模块 (`aluminum`)** *(2026-04-15 新增)*
    *   覆盖铝合金矩形母线（40×4 至 120×10 共15种规格），完整呈现重量、敞露/封闭母线的交流/直流负荷（1/2/3片）。
    *   **表格布局**：多级表头，支持移动端横向滑动。环境温度25℃，数据来源国标参考表。
2.  **铜排查询模块 (`copper`)**
    *   覆盖21种矩形铜排规格，按参考表完整呈现尺寸、重量、敞露/封闭母线的交流/直流负荷（1/2/3片）。
    *   **表格布局**：采用多级表头结构，完全复刻标准表格布局，支持移动端横向滑动查看。数据100%匹配参考表。
    *   仅保留数据展示，纯工具化。
3.  **型材重量查询模块 (`profile`)**
    *   覆盖扁钢（63种）、圆钢、方钢、等边角钢、不等边角钢、槽钢。
    *   下拉框默认显示"槽钢"，支持关键词模糊搜索。
    *   **展示规范**：带全边框表格，展示"规格/型号、每米重量(kg/m)"，关键数据加粗。
4.  **电缆载流量查询模块 (`cable`)** *(2026-04-15 更新)*
    *   支持"三芯/四芯"切换。铜芯型号：VV、YJV、BVV、YJV22；铝芯型号：VLV、YJLV、YJLV22。线径覆盖 **1.5~400mm²**（新增400mm²规格）。
    *   下拉框分组显示铜芯/铝芯型号，铝芯载流量按铜芯约77%系数计算。
    *   **展示规范**：保留空气中/直埋敷设的国标载流量，带全边框表格，核心数据加粗。支持多条件组合筛选。
5.  **架空线载流量查询模块 (`overhead`)** *(2026-04-15 新增)*
    *   支持三种导线类型：LJ（铝绞线）、LGJ（钢芯铝绞线）、JKLGYJ（交联架空绝缘电缆）。
    *   每种类型展示型号、截面、外径、参考重量及 25℃ / 40℃ 两档载流量。
    *   **表格布局**：带全边框表格，导线允许温度：LJ/LGJ 为 70℃，JKLGYJ 为 90℃。
6.  **变压器电流速查模块 (`transformer`)**
    *   覆盖 50~1600kVA，支持 10kV→400V、10kV→700V 变压等级。
    *   **展示规范**：无计算步骤，直接在带全边框的表格中展示高压/低压侧电流结果。
7.  **不锈钢箱体计算模块 (`box`)**
    *   **功能拆分**：分为"展开面积计算"、"底座面积计算"两个独立子标签，参数不混叠。
    *   **展示规范**：输入宽/深/高、板厚、材质、单价等参数，输出面积、理论重量和成本。所有输入和结果均嵌入带全边框的表格，输入限制为正整数。
8.  **电气常用换算模块 (`calc`)** *(2026-04-15 更新)*
    *   支持380V三相系统：kW↔kVA换算及额定电流计算（默认cosΦ 0.8、效率 0.95，支持调整）。
    *   新增 **变压器容量换算**：输入变压器额定容量（kVA）和功率因数，实时输出最大有功功率（kW），公式：P = S × cosΦ。
    *   **展示规范**：核心公式标注国标来源，实时展示结果。
9.  **UPS 功率与续航计算模块 (`ups`)**
    *   **正向计算**（算续航）：输入UPS功率、负载率、电池电压/容量/节数，输出续航时间。
    *   **反向计算**（算电池）：输入目标续航、负载功率，输出推荐电池配置。
    *   **展示规范**：输入/输出嵌入带边框表格，底部标注"仅供现场参考"。

## 3. 全局交互与特性
*   **导航方式**：顶部固定Tab栏（9个模块入口），支持手机端滑动+点击双切换。
*   **移动端适配**：输入框触控面积≥44px，字体最小≥14px，输入框带单位提示、常用参数下拉预设。
*   **纯工具属性**：无广告、无数据埋点、无冗余内容，围绕电气现场查询设计，全面移除"一键复制"功能，专注展示清晰。
*   **路由支持**：通过 `window.location.hash` 记录当前模块，支持链接分享直达特定功能。
*   **数据准确性**：严格符合国标规范与参考数据表。

## 4. 数据维护指南
项目所有数据硬编码在 `index.html` 的 `<script>` 标签内。维护时修改以下常量：
*   `aluminumBusbarData`：铝合金母线（铝排）载流量数据。*(2026-04-15 新增)*
*   `copperData`：多级表头铜排数据。
*   `flatSteelSpecs` / `channelSteelSpecs` 等：型材规格数据。
*   `cableFullData`：电缆数据（含铜芯 VV/YJV/BVV/YJV22 和铝芯 VLV/YJLV/YJLV22，线径至400mm²）。*(2026-04-15 更新)*
*   `overheadLJData` / `overheadLGJData` / `overheadJKLGYJData`：架空线三种型号载流量数据。*(2026-04-15 新增)*
*   `transformerCapacities`：变压器数据。
